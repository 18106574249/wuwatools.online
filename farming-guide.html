Wuthering Waves Farming Guide (3.4)

Waveplate efficiency, best domains, and material farming - updated June 21, 2026.

Waveplate (stamina) is your most valuable daily resource. This guide breaks down where to spend it for maximum efficiency at every account level, and which materials are worth farming vs. buying from the shop.

Material Farming by Type

Character Ascension Gems

Obtained from Forgery Challenge domains (cost 40 waveplate per run). The domain type rotates daily:

• Mon / Thu: Misty Forest (Havoc / Aero gems)
• Tue / Fri: Flaming Remnants (Fusion / Glacio gems)
• Wed / Sat: Cavern of Worms (Electro / Spectro gems)
• Sunday: All domains open

Uses: 1 Sliver → 3 Fragment → 9 Stone → 27 Nugget per character from 1-90.

Talent Materials

Obtained from Simulated Domain (cost 60 waveplate per run). Domains also rotate daily:

• Mon / Thu: Domain of the Forgotten (Skill materials for certain characters)
• Tue / Fri: Domain of the Unyielding (Skill materials)
• Wed / Sat: Domain of the Unbroken (Skill materials)
• Sunday: All domains open

Pro tip: Always check which materials your current team needs before farming. You need ~9 green / 63 blue / 18 purple per character to max all talents.

Boss Materials (Character Ascension)

World bosses drop character-specific ascension materials (cost 60 waveplate each). You need 46 boss materials to fully ascend a character from 1-90.

Key bosses and their materials:
• Crownless - Howling: Last Exile core drop
• Inferno Rider - Roaring: Molten core material
• Tempest Memphisto - Storm: Hecate dimension core
• Dreamless - Phantasmal: Dreamless feather core
• Fallacy of No Return - Mechanical: Abandoned core

Each boss also drops ascension gems and artifacts. Kill the ones your current character needs.

Overworld Farming (Free)

These materials don't cost waveplate and respawn daily or every few days:

• Local specialties (Pecok Flower, Lanternberry, etc.) - each character needs 60 for full ascension
• Enemy drops (Whistle, Ring, etc.) - farming routes reset daily
• Ore nodes (for weapon forging) - check mineral map locations
• Forgery items from puzzles and chests - one-time but worth collecting

Set up a farming route that covers 5-6 material types and run it daily.
